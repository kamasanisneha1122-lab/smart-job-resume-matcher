# Smart Job Resume Matcher 🚀

An AI-powered resume matching system built using Python and Streamlit.

## Features
- Upload Resume PDF
- Paste Job Description
- AI Match Score Calculation
- Missing Skills Detection
- Resume Analysis Dashboard

## Technologies Used
- Python
- Streamlit
- PyPDF2
- Scikit-learn
- NLP (Cosine Similarity)

## How It Works
1. Upload your resume
2. Paste job description
3. Click Match
4. View match score and missing skills
